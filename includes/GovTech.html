Stakeholder Mapping (Data, the Excel & VBA way)

This twin set of files aims to simplify stakeholder mapping for large organisations.

It's all fairly straightforward:

  1. Manager sends input file asking staff to enter their top-10 stakeholders and click "submit".
  2. Manager receives submissions via email.
  3. Manager saves all submissions to a common folder and clicks "join" on the main file.
  4. File joins all submissions into a single database.
  5. Manager clicks "generate map source", file converts everything into .csv files pre-formatted for use by an external mapping software.

Code available via GitHub.
