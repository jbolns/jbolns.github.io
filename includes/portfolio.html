Tools

Atheoretical

[Excel, VBA, Machine Learning]

A set of VBA scripts to rapidly produce supervised ML analyses in Excel. Not for production (no one uses Excel for ML). A demonstration of skills and a proof-of-concept application for, potentially, future web-based versions. That said, the tool could be helpful when you run regressions/supervised ML in Excel periodically and want to save time. Very early stages. Expect bugs.

Link to GitHub repo.

WRD_to

[Word, VBA, HTML]

A VBA script to convert (simple) Word documents into HTML. Helpful when you write a lot in Word (op-eds, journal articles, data analysis briefs, etc.) and need to frequently upload to HTML-driven websites.

Link to GitHub repo.

Data analysis

The cyclistic trilogy

[SQL, Tableau, PowerBI, MicroStrategy, Python, Google Spreadsheets, Excel]

To learn and compare different data analysis tools, I repeated the same analysis (differences in bike usage between casual users and annual members for a Chicago-based bike-share programme) three times, each time with different set of tools and, as a result, a slightly different approach to analysis: (1) SQL → Google Spreadsheets → Tableau, (2) Excel → Power BI, (3) (still in progress) Python → MicroStrategy. I'm not done with the last one.

Link to GitHub repo.
