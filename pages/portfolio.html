The cyclistic trilogy

To learn and compare different data analysis tools, I repeated the same analysis (differences in bike usage between casual users and annual members for a Chicago-based bike-share programme) three times, each time with different set of tools and, as a result, a slightly different approach to analysis: (1) SQL → Google Spreadsheets → Tableau, (2) Excel → Power BI, (3) (still in progress) Python → MicroStrategy. I'm not done with the last one.
