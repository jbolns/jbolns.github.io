Un conjunto de scripts de VBA para producir análisis de ML supervisados ​​en Excel. Disponibles a través de [GitHub](https://github.com/jbolns/atheoretical).

## ¿ML en Excel? ¿Por qué alguien haría eso?
Este proyecto está pensado principalmente como una demostración de habilidades y, potencialmente, una base para futuras versiones.

Dicho esto, si usted tiene datos en Excel, este es el tipo de solución que necesitaría para ejecutar periódicamente ML supervisado sin cambiar de app.
