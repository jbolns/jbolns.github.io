Este proyecto utilizó datos internos de las actividades de una agencia reguladora del Reino Unido para mapear su red internacional. El mapa se utilizó luego como parte de las recomendaciones relacionadas con la estrategia internacional de la Agencia.

Los resultados mapeados de este proyecto estaban destinados al uso interno de la Agencia y, por tanto, no se pueden compartir. Dicho esto, tengo un presentación general del proyecto que se puede compartir a pedido.

El proyecto utilizó Gephi para el mapeo de datos. ¡Muy bonitos los gráficos!
