Una herramienta que combina Python, Javascript y VBA y que permite crear sitios web a partir de distintos tipos de documentos, incluso documentos de Word.

Disponible a través de [GitHub](https://github.com/jbolns/WEIRD).

## Resumen
A pesar de ser un poco difícil de entender, WEIRD puede producir sitios web sorprendentemente buenos y sostenibles.

Dicho esto, WEIRD es bastante difícil de conceptualizar. De hecho, se llama WEIRD porque está programado en capas de diferentes lenguaje (Python, VBA, JS):
- La lógica principal está basada en Python
- Las comunicación con documentos se maneja ya sea en Python o VBA
- Produce sitios web basados ​​en HTML, CSS y JS (estáticos).

## Estado
WEIRD es altamente experimental.

Una vez configurado, WEIRD produce sitios web sorprendentemente sostenibles. El resultado final es realmente impresionante.

Dicho esto, la herramienta es increíblemente difícil de configurar debido a la necesidad de visualizar cómo combinan los diferentes lenguajes de programación.

Publicaré una nueva versión si encuentro una manera de hacerla más fácil de entender.
